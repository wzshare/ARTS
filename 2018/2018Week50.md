## Tip：
### 给一个数组，要求写一个函数，交换数组中的两个元素
```swift
func swap<T>(_ nums: inout [T], _ p: Int, _ q: Int) {
    (nums[p], nums[q]) = (nums[q], nums[p])
}
```

### 实现一个函数，输入是任一整数，输出要返回输入的整数 + 2
