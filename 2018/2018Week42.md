## Tip：

执行
```
(lldb) po NSHomeDirectory()
``` 
可以找到沙盒文件路径

可以使用
```
(lldb)p (void)instrumentObjcMessageSends(YES)
```

或
```
extern void instrumentObjcMessageSends(BOOL);
instrumentObjcMessageSends(YES);
```

开启消息转发的记录，记录文件放在 `/private/tmp/` 中，文件名为 `msgSends-xxx`。
```
cd /private/tmp/
```
