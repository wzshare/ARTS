`@autoclosure` 关键字  
`@autoclosure` 关键字可以自动把表达式转化为闭包，这样可以使代码更加简洁。  
比如实现 `或` 运算时，可以
```swift
func or(value: @autoclosure ()-> Bool, value2: @autoclosure () -> Bool) -> Bool {
  if value() {
    return true
  } else {
    return value2()
  }
}

var res = or(value: 4 > 3, value2: 6 > 2)
print(res)
```
