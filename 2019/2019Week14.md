## Tip：
Swift 中的'＆'运算符不仅可以用来组合协议，还可以用来组合具体的类型和协议，如下代码所示：
```swift
import UIKit
protocol Readable {
    func reload()
}
typealias ReloadableViewController = UIViewController & Readable
class ProfileViewController: ReloadableViewController {
private lazy var tableView = UITableView()
    func reload() {
    // to do something
    }
}
```
